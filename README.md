# Development
## 1. Install dependencies.
```shell
pip install -r requirements.txt
```
And in frontend:
```shell
npm install
```
## 2. Make migrations
```shell
python manage.py makemigrations
python manage.py migrate
```
## 3. Start the server!
```shell
python manage.py runserver
```
And in frontend:
```shell
npm run dev
```
